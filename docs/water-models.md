# Water models

Specify a water model with the `:water_model` suboption of exporter plugins.

| symbol | type | References |
| ------ | ---- | ---------- |
| 3site, spce, tip3p | A typical 3-site model. | [[Jorgensen 1983](references.md#jorgensen-1983)] |
| 4site, tip4p | A typical 4-site model. | [[Jorgensen 1983](references.md#jorgensen-1983), [Jorgensen 1985](references.md#jorgensen-1985)] |
| 5site, tip5p | A typical 5-site model. | [[Mahoney 2000](references.md#mahoney-2000)] |
| 6site, NvdE | A 6-site water model. | [[Nada 2003](references.md#nada-2003)] |
| 7site | A seven-site water model. | [[Zhao 2019](references.md#zhao-2019)] |
| ice | TIP4P/Ice water model. | [[Abascal 2005](references.md#abascal-2005)] |
| physical_water | Physical model of water; Oxygen atom is on the lattice point. | [[Jorgensen 1983](references.md#jorgensen-1983)] |
|  | (Undocumented) | |

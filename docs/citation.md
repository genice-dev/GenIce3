# Citation

If you use GenIce in your work, please cite as described in [CITATION.cff](https://github.com/genice-dev/GenIce3/blob/main/CITATION.cff) or the papers below.

The algorithms for generating depolarized, hydrogen-disordered ice are described in:

> M. Matsumoto, T. Yagasaki, and H. Tanaka, "GenIce: Hydrogen-Disordered Ice Generator", _J. Comput. Chem._ **39**, 61-64 (2017). [DOI: 10.1002/jcc.25077](http://doi.org/10.1002/jcc.25077)

}
